# BiType_byJan_A_Wojtas
BiType is a speculative typeface system constructed entirely on binary logic. Instead of standard bezier contours, every letter, number, and punctuation mark is defined inside a 4x4 space. Each cell holds either a 1 or 0. The typeface makes digital encoding visible, establishing a visual medium where letterforms double as raw data.
